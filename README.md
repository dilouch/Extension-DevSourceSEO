## 🚀 Présentation

**DevSourceSEO** est l'outil indispensable pour analyser instantanément les performances SEO et la structure technique de n'importe quelle page web directement depuis votre navigateur.

---

## 🔥 Pourquoi utiliser DevSourceSEO ?

Cette extension simplifie l'audit SEO technique pour les développeurs et marketeurs en extrayant les données critiques sans avoir à fouiller dans l'inspecteur d'éléments.

### 📋 Fonctionnalités clés :
* **Analyse de contenu :** Vérification instantanée des balises `<title>` et `<meta description>`.
* **Hiérarchie Sémantique :** Visualisation claire de la structure des titres (`H1` à `H6`).
* **Indexation :** Vérification des balises `canonical` et des directives `robots` (noindex, nofollow).
* **Réseaux Sociaux :** Aperçu des balises Open Graph (Facebook) et Twitter Cards.
* **Images :** Détection des balises `alt` manquantes.

---
## 📥 Installation

### Via le Chrome Web Store
1. Rendez-vous sur la page officielle : [**DevSourceSEO sur Chrome Store**](LIEN_VERS_TON_EXTENSION).
2. Cliquez sur **Ajouter à Chrome**.
3. Épinglez l'extension en cliquant sur l'icône "Puzzle" 🧩 en haut à droite de votre navigateur pour un accès rapide.

---

## 🛠️ Stack Technique

L'extension est conçue pour être légère et rapide :
* **Langages :** HTML, CSS, JavaScript.
* **Architecture :** Manifest V3.
* **Performance :** Zéro dépendance externe pour garantir une exécution instantanée.

---
## 💡 Utilisation

1. Naviguez sur le site que vous souhaitez analyser.
2. Cliquez sur l'icône **DevSourceSEO**.
3. Parcourez les différents onglets pour obtenir les informations détaillées du code source.
